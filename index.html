03 · Process

How a project moves through my hands

The same four steps, whether it's a mobile app or a landing page.

01

Research

Understand who's using this and what's actually slowing them down, before opening Figma.

02

Wireframe

Map the flow low-fidelity first, so the structure is right before it's pretty.

03

Prototype

Build the real screens in Figma and click through them like a user would.

04

Build

Turn the final design into responsive, working code — or hand it off ready to build.
